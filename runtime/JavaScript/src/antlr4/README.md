# JavaScript target for [ANTLR 4][antlr]

## Installation
```bash
npm i -S antlr4
```

## Tests
This runtime has been tested in Node.js, Safari, Firefox, Chrome and IE.

[For more information on ANTLR][antlr]
[For more information on using ANTLR in JavaScript][jsdocs]
